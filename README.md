# Windows Registry API Research

## Overview

This project is a Windows Internals learning exercise focused on understanding the Windows Registry and the Win32 Registry API. The objective is to study how Windows applications interact with the registry, how configuration data is stored, and how registry-related APIs are used in native Windows development.

## Learning Objectives

- Understand the Windows Registry structure
- Explore Win32 Registry APIs
- Learn registry key and value management
- Study Windows configuration mechanisms
- Gain experience with C programming on Windows
- Improve understanding of Windows Internals

## Topics Covered

- Windows Registry
- Registry Hives
- Registry Keys and Values
- Win32 Registry APIs
- Windows API Programming
- C Programming
- Windows Internals

## Technologies

- C
- Win32 API
- Windows SDK
- Visual Studio Code

## Purpose

This repository was created for educational purposes to improve knowledge of Windows Internals, Windows API programming, and registry concepts in a controlled learning environment.
